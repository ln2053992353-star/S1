### 3.3 附带修复：搜索页 PubChem 数据缺失

在验收过程中发现，8000 端口（Django 搜索页）进行动态搜索时，返回结果中 **PubChem 数据丢失**。

**原因：** `search_engine/views.py` 的 `_build_formatter_data` 函数在构造搜索结果 JSON 时，未从数据库 `YeastPubChemData` 表中读取 PubChem 数据，`iupac_name` 被硬编码为空字符串。

**修复：** 新增 `_collect_pubchem()`、`_build_enriched_description()` 等辅助函数，在搜索结果中自动附加 PubChem 数据块，前端正确渲染。

## 四、运行方式

```bash
# 独立前端模块（纯展示，读 JSON）
cd frontend
python -m http.server 8080
# 浏览器打开 http://localhost:8080

# 原有 Django 搜索页（无需改动）
python manage.py runserver 0.0.0.0:8000
# 浏览器打开 http://localhost:8000
```

两个端口各自独立运行，互不影响。
